# 🧠 Lógica Aplicada em Python

## 📌 Sobre o projeto

Este projeto reúne exercícios desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas (ADS), com foco na aplicação prática da lógica de programação utilizando Python.

Os exercícios abordam problemas do dia a dia, utilizando operações matemáticas e manipulação de dados.

## 🛠️ Tecnologias utilizadas

* Python

## 🎯 Funcionalidades

* Resolução de problemas matemáticos
* Manipulação de entradas do usuário
* Aplicação de lógica básica

## 📝 Desafios propostos

1. Calcular salário com acréscimo de 5% de comissão
2. Calcular velocidade média (distância / tempo)
3. Converter dólar para real
4. Calcular valor total com 10% de gorjeta
5. Converter temperatura de Celsius para Fahrenheit e Kelvin
